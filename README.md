# Wireshark Network Traffic Analysis

Beginner-friendly network traffic analysis project using Wireshark.

## Project Goal
The goal of this project is to analyze basic network traffic and understand common protocols such as DNS, HTTP, and ICMP.

## Tools Used
- Wireshark
- Kali Linux / Windows
- Browser
- Command Prompt / Terminal

## Analysis Topics
- DNS traffic analysis
- HTTP request and response analysis
- ICMP ping traffic
- Basic packet inspection

## What I Learned
- How to capture network packets
- How to filter traffic in Wireshark
- How DNS queries work
- How HTTP communication works
- How to identify source and destination IP addresses
